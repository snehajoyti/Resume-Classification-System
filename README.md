# 🚀 Resume Classification System – Smart Hiring Catalog

Resume Classification System transforms modern recruitment into a fast, explainable, and data-driven workflow built for smart hiring teams. Designed to solve the challenge of screening massive resume volumes, it combines Natural Language Processing, machine learning, and automation to identify the most relevant candidates with speed and fairness.

The pipeline begins with seamless resume ingestion, supporting PDF and DOCX uploads for flexible candidate intake. Using spaCy and NLTK, each document is processed through tokenization, lemmatization, and named entity recognition to extract structured details such as candidate name, skills, education, work history, and professional summary. This cleaned information is stored in a SQL database, enabling fast retrieval, filtering, and scalable management.

## 🧠 Intelligent Classification Engine
At its intelligence core, the classification engine converts resumes and job descriptions into machine-readable vectors using TF-IDF and semantic embeddings. Cosine similarity and ranking algorithms then measure contextual alignment, assigning every applicant a transparent match percentage. This allows recruiters to instantly take action with clear outcomes: ✅ Select, ❌ Reject, or ⭐ Shortlist.

## 🎯 Recruiter-Centric Interface
The interface is crafted for recruiter productivity and trust. Parsed candidate insights appear alongside original resume content, making evaluations intuitive and fast. Advanced filters support prioritization by technical skills, experience level, role relevance, and domain expertise. CSV export functionality ensures smooth integration with HR workflows, reporting dashboards, and talent pipelines.

## 🔍 Transparent, Scalable, and Impactful
A defining strength of the system is explainability. Every recommendation includes clear reasoning behind the assigned score, reducing black-box decisions and improving recruiter confidence. By automating repetitive screening tasks, minimizing bias, and accelerating decision-making, it empowers hiring teams to focus on building diverse, high-performing organizations.

From data ingestion and preprocessing to model training, evaluation, deployment, and results export, this project demonstrates a complete end-to-end AI hiring solution. Perfect for recruiters, HR teams, career platforms, and academic showcases, it highlights how responsible AI can modernize talent acquisition with transparency, scalability, usability, and measurable workforce impact across enterprises, startups, universities, and career counseling ecosystems. It also supports customizable skill taxonomies, role-specific scoring logic, and future-ready model retraining workflows, ensuring adaptability as hiring trends evolve. With its balance of precision, fairness, and operational efficiency, it stands as a production-ready smart hiring catalog solution today.
